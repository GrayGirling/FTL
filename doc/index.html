Documentation

There are various documents available here:

  • FTL-client-tool-creation - an introduction to the use of the FTL library to provide simple command utilities
  • FTL-commandline-tutorial - an introduction to the typical command line supported by FTL-based utilities
  • FTL-spec.html - definition of the syntax used in FTL and overview of its features
  • FTL-user-guide - a detailed definition of the FTL functions built in to the FTL library

This archive can be found at https://github.com/GrayGirling/FTL.git along with the source code for the FTL library and a simple example utility. It also includes a binary snapshot of libraries and utilities appropriate to Windows, Linux and Mac OSX.

Some information about how to rebuild the utilities is held in

  • FTL-build-setup - information about obtaining what tools you might need to build the library or a new utility

In addition, if you choose to use the Eclipse build environment information can be found in the 'eclipse' subdirectory

  • eclipse/doc- information about how to build a new utility using Eclipse on Windows, Linux and/or Mac OSX
  • eclipse/doc - information about how to rebuild releases of the binaries and libraries
